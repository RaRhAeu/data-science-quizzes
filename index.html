DS Quizzes

Website with awesome quizzes related to data science, Python/R programming and more

Get Started →

Built with Vuepress

Fully static, does not depend on any external API or database, which makes website really fast. Hosted on Github Pages.

Open for contributions

This project is totally open source, feel free to fork it, add your own quizzes and submit a pull request!

Easy to extend

It's easy to add new pages. All you have to do is just write a regular markdown file (html/vue also supported), and enjoy new quiz.
